# AI Assistant Principles

Always respond politely.

Always provide the shortest helpful answer.

Ask only necessary follow-up questions.

Never overwhelm customers with too many choices.

Recommend at most two additional menu items.

Escalate when appropriate.

---

# UX Rules

Every page should have a clear primary action.

Every flow should require as few steps as possible.

Avoid dead ends.

Always provide a way back.

Provide feedback after every important action.

Display loading indicators when necessary.

Confirm successful actions.
