Version: 1.0

# Executive Summary

Inzane's Food is a premium Asian comfort food restaurant website designed to provide customers with an elegant online experience while showcasing modern restaurant technology.

The website should feel premium, welcoming, responsive, and future-ready.

Beyond serving as a restaurant website, the platform is intended to demonstrate AI-powered restaurant automation capabilities that can later be expanded into a complete Restaurant Management System.

---

# Project Vision

Build one of the best-looking modern restaurant websites that combines premium branding, intuitive user experience, online ordering, reservations, and AI-powered customer service.

The project should serve as:

• Restaurant Website

• Online Ordering Platform

• Reservation System

• AI Assistant

• Restaurant CRM

• AI Automation Portfolio

# Final Instructions for Lovable

Read every document inside the `/docs` folder before generating any code.

Treat all documentation as the official source of truth.

Do not redesign the branding.

Do not invent new colors.

Do not replace the uploaded logos.

Use uploaded food photography whenever available.

Use premium placeholders for missing food images.

Build reusable React components.

Optimize for performance and SEO.

Generate production-ready code following modern best practices.

The website should feel comparable in quality to leading modern restaurant websites while remaining unique to the Inzane's Food brand.
